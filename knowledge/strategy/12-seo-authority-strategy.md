## SEO Philosophy

FortunaDelClub's SEO strategy is built on three pillars:
1. **Topical Authority** — Become the most comprehensive resource for Brazilian betting/casino information
2. **Search Intent Match** — Every page must perfectly match what the user is looking for
3. **Technical Excellence** — Fast, mobile-friendly, well-structured site that Google can easily crawl and index

---

## Target Keyword Categories

### Tier 1 — High Priority (Core Keywords)
These are the most competitive but highest-value keywords:
- "apostas esportivas" and variations
- "cassino online brasil" and variations
- "melhores casas de apostas" and variations
- "bonus apostas" and variations
- Platform-specific: "bet365 brasil", "betano brasil", etc.

### Tier 2 — Medium Priority (Long-tail Keywords)
Lower competition, high conversion intent:
- "como apostar no [sport] online"
- "como depositar via pix [platform name]"
- "bonus de boas-vindas [platform name]"
- "e seguro apostar em [platform name]"
- "como sacar dinheiro [platform name]"

### Tier 3 — Supporting Keywords (FAQ and Educational)
Question-based, featured snippet opportunities:
- "como funciona aposta esportiva"
- "o que e odds"
- "apostas esportivas sao legais no brasil"
- "como funciona bonus de cassino"

## On-Page SEO Requirements

### Every Page Must Have
- **Title tag:** Primary keyword + brand name, under 60 characters
- **Meta description:** Compelling, includes keyword, 150–160 characters
- **H1:** One per page, includes primary keyword
- **H2/H3 structure:** Logical hierarchy, includes secondary keywords
- **First paragraph:** Primary keyword within first 100 words
- **Image alt text:** Descriptive, includes keyword where natural
- **Internal links:** Minimum 2 per page (see 07-internal-linking-strategy.md)
- **Schema markup:** Appropriate schema for content type (Review, FAQ, Article, etc.)
- **Canonical tag:** Prevent duplicate content issues
- **Last updated date:** Visible on page

### Schema Markup by Content Type
| Content Type | Schema Type |
|-------------|-------------|
| Platform reviews | Review, Organization |
| How-to guides | HowTo |
| FAQ pages | FAQPage |
| News articles | NewsArticle |
| Homepage | WebSite, Organization |

## Technical SEO Requirements

### Site Speed
- Target: Core Web Vitals "Good" rating on mobile
- Images: Compressed and served in WebP format
- Lazy loading: Implement for images below the fold
- Caching: Browser and server-side caching enabled
- CDN: Use a CDN for static assets

### Mobile Optimization
- Responsive design — mobile-first
- Touch-friendly buttons (minimum 44x44px tap targets)
- No intrusive interstitials on mobile
- WhatsApp button easily tappable on mobile

### Crawlability
- XML sitemap submitted to Google Search Console
- Robots.txt properly configured
- No important pages blocked from crawling
- Internal links use crawlable HTML (not JavaScript-only)
- 301 redirects for any changed or removed URLs

### Structured Data
- Implement JSON-LD structured data (not microdata)
- Validate with Google's Rich Results Test
- Monitor for structured data errors in Search Console

## Authority Building Strategy

### Content Depth
- Pillar pages should be comprehensive (2,000+ words)
- Cover all sub-topics within a pillar
- Answer all common questions on the topic
- Include original analysis or community insights where possible

### E-E-A-T Signals (Experience, Expertise, Authoritativeness, Trustworthiness)
Google evaluates gambling sites heavily on E-E-A-T:
- **Experience:** Reference real platform testing and community feedback
- **Expertise:** Demonstrate deep knowledge of betting mechanics, odds, regulations
- **Authoritativeness:** Build topical authority through comprehensive coverage
- **Trustworthiness:** Maintain all compliance elements (disclaimers, legal pages, transparency)

### Link Building (Off-Page)
- Focus on earning links through quality content
- Target Brazilian betting/sports media for mentions
- Community engagement (WhatsApp, forums) can drive indirect signals
- Avoid buying links or participating in link schemes

## Google Search Console Monitoring

Monitor weekly:
- Impressions and clicks for target keywords
- Pages losing traffic (may need content refresh)
- Core Web Vitals issues
- Coverage errors (crawl issues, 404s)
- Manual actions (check monthly)

## Local/Regional SEO

- All content targets Brazilian Portuguese speakers
- Use Brazilian Portuguese in hreflang tags: hreflang="pt-BR"
- Reference Brazilian-specific context (PIX, Brasileirao, Brazilian regulations)
- Google Business Profile: Not applicable (online-only site)
