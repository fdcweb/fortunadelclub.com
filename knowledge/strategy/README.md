## Key Principles (Quick Reference)

- **Never guarantee winnings** — FortunaDelClub is an informational/community site, not a gambling operator.
- **Always include responsible gambling disclaimers** on relevant pages.
- **Maintain editorial independence** — content must appear unbiased and research-based.
- **WhatsApp is the primary CTA** — all pages should funnel toward WhatsApp contact.
- **SEO-first content** — every page must target specific keywords and serve search intent.
- **Legal pages must be kept current** — review Terms, Privacy Policy, and Disclaimer quarterly.
